Для начала работы откройте VangersControl.exe и настройте управление под себя. поднятый большой палец вверх одной руки деактивирует любые нажатия.

Управление:
+-------------------+----------------------------------------------------+
| Клавиша           | Жест / Условие                                     |
+-------------------+----------------------------------------------------+
| ← (влево)        | Поворот кистей влево                               |
+-------------------+----------------------------------------------------+
| → (вправо)       | Поворот кистей вправо                              |
+-------------------+----------------------------------------------------+
| ↑ (вверх)         | Расстояние от запястья до кончика среднего пальца |
|                   | меньше порога (сжатая ладонь)                     |
+-------------------+----------------------------------------------------+
| ↓ (вниз)          | Расстояние больше порога (раскрытая ладонь)        |
+-------------------+----------------------------------------------------+
| W                 | Средняя высота запястий выше порога                |
|                   | (руки подняты) — приоритетная клавиша             |
+-------------------+----------------------------------------------------+
| A (альт-режим)    | Правая рука щиплет, левая нет                      |
+-------------------+----------------------------------------------------+
| D (альт-режим)    | Левая рука щиплет, правая нет                      |
+-------------------+----------------------------------------------------+
| LShift            | Кончик мизинца дальше кончика безымянного          |
|                   | (любая рука)                                      |
+-------------------+----------------------------------------------------+
| E                 | Обе руки одновременно: кончик указательного        |
|                   | касается основания большого пальца,               |
|                   | и стрелка ↓ не нажата                             |
+-------------------+----------------------------------------------------+
| Space             | Расстояние между запястьями меньше порога          |
|                   | (руки сведены)                                    |
+-------------------+----------------------------------------------------+
| Авто-↑            | Расстояние между запястьями больше порога          |
| (Forward-Only)    | (руки разведены) — зажимает ↑ и блокирует         |
|                   | все остальные клавиши                             |
+-------------------+----------------------------------------------------+

Приоритеты:
+-------------------+----------------------------------------------------+
| W                 | Высший приоритет — блокирует все остальные клавиши |
| Авто-↑            | Блокирует все, кроме W (но W не может быть         |
| (Forward-Only)    | одновременно активна, так как руки разведены)      |
+-------------------+----------------------------------------------------+
